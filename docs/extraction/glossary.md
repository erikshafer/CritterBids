# Ubiquitous-Language Glossary

Populated in Phase 3 of the extraction. Each entry will carry a definition in the auction domain and a citation to where the term is honored in code (and, where applicable, a cross-reference to `gaps-and-drift.md` for terms whose code and intent disagree).

Seed terms (verified during Phase 3):

- Listing
- Sale / Flash Session
- Starting Bid
- Reserve
- Hammer Price
- Final Value Fee
- Extended Bidding
- BidderId
- ListingSold
- ListingPassed
- BiddingClosed
- ReserveMet
- ReserveCheckCompleted
- Buy It Now
- Proxy Bid

Linguistic conventions (captured as glossary entries in Phase 3):

- "paddle" prohibition
- "no Event suffix" event-naming rule
- `BiddingClosed` ≠ `ListingSold` / `ListingPassed` (mechanical fact vs business outcome)
- `ReserveMet` ≠ `ReserveCheckCompleted` (real-time UX signal vs binding financial check)
- `ListingEndedEarly` ≠ `ListingWithdrawn` (seller-initiated vs ops-initiated)
